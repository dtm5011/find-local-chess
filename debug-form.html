Debug Form Submission

Open browser console and navigate to: http://localhost:4321/find-local-chess/contribute

Debug Steps:

  1. Open the contribute form
  2. Open browser developer tools (F12)
  3. Go to Console tab
  4. Fill out the form
  5. Try to submit and check for any JavaScript errors
  6. Check Network tab to see if the API request is being made

Common Issues to Check:

  • JavaScript errors in console
  • Network request failing (check Network tab)
  • Form validation errors
  • API endpoint returning errors
